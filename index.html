The difference between (==) and (===) operators.

These equal sign operators are so confusing when first working with them. The double equal (==) sign checks for value equality only. This double equal sign is different in the sense that, it inherently does type coercion. This means that, before checking the values, it converts the type of variables to match each other. However, the triple equals (===) does not perfom type coercion.
